# Anchor & Oak — The Father's Table

A faith formation web app for Christian fathers raising children aged 4–7.

## Files

- `index.html` — the entire application (single file)
- `vercel.json` — Vercel deployment config

## Deployment

1. Push both files to the **root** of this GitHub repository
2. Connect repo to Vercel — it will auto-deploy on every push
3. No build step required — static HTML

## Audio Narration (ElevenLabs)

1. Get a free API key at https://elevenlabs.io
2. Open the live site → tap ⚙️ Settings
3. Paste your key → tap 💾 Save Key → tap 🔊 Test

## PayPal Payments

Edit `index.html` and fill in your PayPal details in the config block near the top of the `<script>` tag:
- `PAYPAL_CLIENT_ID`
- `PAYPAL_PLAN_MONTHLY`
- `PAYPAL_PLAN_ANNUAL`
